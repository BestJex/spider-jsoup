## 前言

`spider-jsoup`项目属于爬虫入门项目，仅可爬取静态网页，动态网页无法抓取

## 项目介绍

`spider-jsoup`项目是基于SpringBoot+jsoup的入门爬虫系统，包括 接口爬、定时爬、多线程爬
